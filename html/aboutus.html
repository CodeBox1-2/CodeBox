What is the purpose of Code Box?

#1

Dissemination of science The first goal of this academy is to spread knowledge and awareness for free in the Arab community, as our nation will not advance without knowledge, and we have begun with ourselves.

#2

Prevent exploitation We have set the dates for educational courses in a value-added manner, and that is why we create our courses for free as a community to be kind to people.

#3

Programming specialty Programming is now a tool for countries' progress, and whoever owns technology now has progress, so we decided to specialize in spreading programming in Arab society in order to advance.

What is Code Box?

Code Box is an entity that was established with the aim of teaching programming online for free. There are many educational courses, whether written, visual, or both, and there is a question bank in most programming languages ​​that helps the student in understanding his programming subject greatly. Here there are many specializations such as web front-end and Pack and also has programming basics that aim to establish the student in programming and other important disciplines.
