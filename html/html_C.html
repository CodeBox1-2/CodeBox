Description

ما هي HTML ؟ كلمة HTML هي إختصار لجملة Hyper Text Markup Language و هذا يعني أنها لا تعتبر لغة برمجة و ببساطة هي لغة تستخدم لبناء عناصر صفحة الويب حيث يمكنك من خلالها وصف كل عنصر و ما يتحويه بمعنى انك تستطيع ان تصف ان هذا العنصر عنوان للصفحة و ان العنصر الاخر هو فقرة و يمكنك اضافة صور و مقاطع فيديو و صوت و جداول و ازرار و غيرها لبناء موقعك باختصار هذه اللغة تستخدم لبناء هيكل صفحات الويب. ماذا ستتعلم في هذه الدورة؟ ستتعلم لغة html & html5 بالطبيقات العملية لكي تكون قادر على بناء هيكل ويب كامل و تحديد و معرفة جميع عناصر html و التفريق بينهم و متى يتم استخدام كل واحد فيهم. هل لغة HTML كافية لعمل موقع كامل؟ بالطبع فان لغة html قوية ولا غنى عنها فلن تستطيع بناء صفحات ويب بدون هذه اللغة و لكن html مختصة فقط بتصميم هيكل موقعك و تحديد عناصره و لكن لنطرح سؤال هل يكفي لك ان تصمم هيكل الموقع؟ بالطبع لا , فعلى سبيل المثال المهندسين عندما يشرعون ببناء منزل يقوموا بتصميم هيكل المنزل الذي لا غنى عنه و لكن بالاضافة الى الهيكل فانهم يقوموا بدهانه و تنسيقه و توصيل الكهراء و المياة و غيره كذلك الويب فان هذا الهيكل الذي صنعته بال html يجب تنسيقه و ضبط ابعاده و اضافة الوان و صور ممتعة و ترتيب العناصر داخل هذا الهيكل ليصبح لدينا تصميم منسق و كل هذه التنسيقات نصنعها بال css لهذا تجد ال css مرتبطة بال html دائما لانهم يكملوا بعضهم , و بالاضافة الى ذلك فيتم استخدام لغة جافاسكريبت لاعطاء الحركة الى موقعك بأختصار لكي تستطيع بناء صفحات ويب فانت بحاجة الى تعلم ثلاث لغات HTML & CSS & JAVASCRIPT. ماذا احتاج لكي ادخل الى هذه الدورة؟ حرفياً انت لا تحتاج الى تعلم اي شئ فهذه الدورة هي بدايتك الفعلية و الخطوة الاولى لك في عالم الويب.
